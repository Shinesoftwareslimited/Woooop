VISION-V1.2-A

A Single-Tier, High-Precision Object Detection Framework

Architecture Overview

VISION12n is a high-performance, single-tier object detection model developed by Vigor Infotech. It utilizes a custom convolutional backbone and a modular detection head optimized for real-time multi-class object detection across diverse environments.

  • Backbone built with Conv, C3, and A2C2f modules
  • 5-stage spatial reduction pipeline (P1 to P5)
  • Fusion of high, mid, and low-level features
  • Custom Segment head for final detection

Backbone and Feature Flow

  • P1: Conv layer (64 filters)
  • P2: Conv layer (128 filters)
  • P3: C3 ×2 (256 filters)
  • P4: A2C2f ×4 (512 filters)
  • P5: A2C2f ×4 (1024 filters)
  • Feature pyramid fusion via upsampling and Concat

Detection Head

  • Upsampling with nearest neighbor strategy
  • Skip connections from backbone layers P3, P4, P5
  • Multi-scale detection using Segment module
  • Outputs bounding boxes + class predictions

Model Specifications

Total Parameters: ~30M
FLOPs: ~145.1 GFLOPs
Depth: 510 layers
Object Classes: 10
Input Size: 512×512 (recommended)
Modules: Conv, C3k2, A2C2f, Segment
Activation: SiLU
Upsampling: Nearest neighbor (2x)
Framework: PyTorch
